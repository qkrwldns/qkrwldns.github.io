## 설치

- jdk 설치
- path 등록 
- javac --version을 cmd에 입력시 버전이 출력되어야 정상동작.  


## 단축키

- syso 하고 컨트롤 + 스페이스 바 : 자동완성
- 컨트롤 + n : 생성
- 컨트롤 + f11 : 실행
